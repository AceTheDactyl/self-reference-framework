This directory stores raw and processed outputs from the validation scripts.

- sr1_fixedpoint_results.json — output of `theorem_sr1_fixedpoint.py`
- sr3_fibonacci_convergence.json — output of `fibonacci_convergence.py`
- sr6_percolation_data.csv — sweep of p vs percolation probability
- validation_summary.json — aggregated run summary

All files are generated by `code/run_all_validations.sh`.
