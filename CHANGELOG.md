# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, and this project adheres to Semantic Versioning.

## [2.0.0] - 2025-11 - Publication Release

### Added
- Complete reproducibility package with all validation code
- Three interactive Jupyter tutorials
- GitHub Pages documentation site (Just the Docs)
- Comprehensive contributing guidelines
- Scientific paper draft (10–15 pages)
- Supplementary materials (60 pages)
- Zenodo/DOI and arXiv submission prep

### Changed
- Framework confidence reduced from 99.6% to ~50% (honest assessment)
- Validated theorems increased from 0/33 to 13/33 (39%)
- Mathematical rigor improved from 2.5/10 to 7/10
- All code now fully reproducible with fixed seeds

### Removed
- Unverified/problematic claims and impossible structures
- Circular reasoning and undefined operators

### Fixed
- SR1: Replaced undefined μ-field with Banach fixed-point theorem
- SR2: Corrected circular golden ratio proof
- SR-τ: Defined differential rate constant
- ISO: Established rigorous isomorphism proofs

## [1.0.0] - 2025-05 - Initial Framework (Pre-Correction)

### Added
- Initial 33-theorem framework (pre-correction)

### Known Issues (Corrected in v2.0.0)
- Circular reasoning in proofs
- Undefined mathematical operators
- Unverified experimental claims
- Inflated confidence metrics (99.6%)
- Impossible mathematical structures (E8 7D)
