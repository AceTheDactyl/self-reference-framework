## About
This project transforms a 33-theorem self-reference framework from speculative claims to a rigorous, falsifiable, and reproducible body of work via a 5-phase pipeline.

## How To Reproduce
- Install dependencies listed in `REPRODUCIBILITY_PACKAGE/code/requirements.txt`.
- Run: `bash REPRODUCIBILITY_PACKAGE/code/run_all_validations.sh`
- Figures: `python REPRODUCIBILITY_PACKAGE/figures/figure_generation.py`
