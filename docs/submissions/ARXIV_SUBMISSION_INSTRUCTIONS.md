# arXiv Submission Instructions

Submission package: arxiv_submission_v2.0.tar.gz

1. Go to https://arxiv.org/submit and start a new submission.
2. Upload the tarball; arXiv extracts the files.
3. Fill metadata (title, authors, abstract, categories).
4. Verify compiled PDF preview (or attach an external PDF if LaTeX not compiled).
5. Submit and record the assigned arXiv ID.
6. Update README and paper with arXiv ID.
